# Publicação de documentos via API pública
---

Neste exemplo, veja como publicar documentos através da API pública fluig, utilizando JavaScript.

A pasta de publicação está definida no arquivo upload_file_rest.js

Saiba mais sobre as APIs do fluig acessando a página [Fluig API](http://tdn.totvs.com/x/KAFlDw).
Você também pode consultar as APIs do fluig clicando [aqui](http://api.fluig.com/).
