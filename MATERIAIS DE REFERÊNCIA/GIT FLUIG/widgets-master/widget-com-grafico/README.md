
# Widget com visualização de gráficos


## Caso de uso
A widget permite a visualização de gráfico do tipo pizza, radar e donut buscando dados de usuários, grupos e papéis do fluig.

## Técnicas abordadas
- Implementação de widget;
- Consulta dos datasets 'colleague', 'colleagueGroup' e 'workflowColleagueRole';
- Criação de gráficos utilizando a biblioteca fluig Style Guide.

## Requisitos para o funcionamento do desenvolvimento
- Nenhum requisito específico é necessário.
