# Dados adicionais do usuário: Campos Editáveis
---

Neste exemplo veja como criar/editar dados adicionais do usuário.
Para exemplificar melhor, foram criados três tipos de campos: Campo de texto simples, Caixa de seleção (Combo) e um campo textarea.

Acesse a página de Ajuda e entenda o objetivo dos campos já existentes na tela de cadastro do [Usuário](http://tdn.totvs.com/x/c4H5DQ).
