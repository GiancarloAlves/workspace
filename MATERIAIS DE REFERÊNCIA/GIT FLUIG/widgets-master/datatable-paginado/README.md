
# Datatable paginado


## Caso de uso
A widget lista usuários do fluig através de um datatable padrão do fluig Style Guide, implementando a paginação e filtro de resultados.

## Técnicas abordadas
- Implementação de widget;
- Consulta na api fluig consumindo o endpoint '/api/public/wcm/user/findUsingFilter';
- Utilização do datatable padrão fluig Style Guide com paginação e filtro.

## Requisitos para o funcionamento do desenvolvimento
- Nenhum requisito específico é necessário.
